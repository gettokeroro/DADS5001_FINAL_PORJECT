2. Scoring function (Python module)

Implement TF-IDF inspired scoring:
TF = 1 ถ้า user ติ๊ก
IDF = log(N_diseases / N_diseases_with_this_symptom)
Score(disease, user_symptoms) = Σ TF × IDF + bonus ถ้า match ≥ 80% ของ symptoms ที่ disease นั้นต้องการ
รับ list ของ symptoms ที่ user ติ๊ก → คืน top-3 disease + score + confidence
3. Eval mini-set (5-10 cases)

"ติ๊ก [chest_pain, sweating, breathlessness]" → Heart attack top-1
ใช้ทดสอบว่า scoring function ดีพอไหม → เป็น baseline สำหรับ AI mode เปรียบเทียบ

วิธีหา "co-symptoms" — Data-driven
จาก initial symptoms ของ user → run TF-IDF เบาๆ ก่อน → ได้ top-5 candidate diseases → query SQL หา อาการอื่นๆ ที่มีใน candidates เหล่านั้น แต่ user ยังไม่ได้บอก:

sql
SELECT l.symptom, SUM(l.freq * d.score) AS relevance
FROM disease_symptom_long l
JOIN candidate_diseases d ON l.disease = d.disease
WHERE l.symptom NOT IN (user_already_mentioned)
  AND l.freq >= 0.5
GROUP BY l.symptom
ORDER BY relevance DESC
LIMIT 7
→ ได้ symptoms ที่ "co-occur" กับโรคที่น่าจะเป็นจริง · ไม่ใช่สุ่ม

(ข) Confidence-based — ใช้ softmax probability
